# Design assets

Brand assets and UI mockups for the BIM Hive Marketplace rebuild. Treat these + `../style.md`
as the design source of truth.

## Brand
- `logo.png` — gold hexagonal BIMHIVE monogram (the "B/H" mark). Source logo for the build.
- `hero-background.png` — homepage hero image: the gold monogram over a light architectural
  wireframe, with empty space on the left for the headline text.

## Storefront mockups
- `mockup-storefront-home.png` — homepage: hero, trust-badge row, category sidebar, featured
  products grid, popular collections.
- `mockup-storefront-product-detail.png` — product detail: media gallery, buy-box (license
  tiers), tabs (Overview, Features, Reviews, Compatibility, Documentation, Support), What's New,
  ratings breakdown, publisher card.
- `mockup-storefront-checkout.png` — checkout: order, email, payment method (card/PayPal/ACH),
  sticky order-summary panel, Stripe branding, trust reinforcements.

## User account mockups
- `mockup-account-profile.png` — account Profile page + account nav (Overview, Licenses,
  Subscriptions, Orders & Invoices, Payment Methods, Profile, Security, Notifications; Support:
  Support Tickets, Downloads).
- `mockup-account-licenses.png` — account Licenses page: stat cards (Active / Expiring / Expired
  / Total Spent), filter tabs, license table with per-row download + Manage actions.

## Admin portal mockups
- `mockup-admin-products-list.png` — admin Products list: status tabs, stat cards, product table,
  recent tickets + top-products panels; full admin sidebar (Overview, Products & Content,
  Support, Settings).
- `mockup-admin-add-product.png` — admin Add/Edit Product form: tabs (Product Information, Media &
  Previews, Pricing & License, Files & Downloads, Compatibility, SEO & Settings) + publishing rail
  (status, visibility, product type, additional settings).
